# consume-imgur

Imgur gallery preview app. Demo [here](https://eurowings-c7d44.web.app).

Requirements:
- Git
- Node v.8
- npm v.6

To run the app locally:


```bash
git clone https://github.com/tonym3nsa/consume-imgur
cd consume-imgur
npm install
npm start
open http://localhost:3000 in your browser of choice
```
